# Лечебное меню

Небольшой фронтенд-проект (чистый HTML/CSS/JS), который помогает по тексту рекомендаций врача собрать визуальное лечебное меню на день, список покупок на неделю и экспортировать результат в JSON.

## Возможности

- Приветственный лендинг с быстрым стартом и FAQ.
- Загрузка рекомендаций из файлов (`.txt`, `.md`, `.csv`, `.json`).
- Автоанализ ограничений и типа питания по ключевым словам.
- Генерация меню на день с заменой блюд и фото-подачей.
- Автопостроение списка покупок на неделю.
- Ручное сохранение и восстановление черновика в браузере.
- Экспорт результата в JSON.

## Стек

- HTML5
- CSS3
- Vanilla JavaScript (без фреймворков и сервера)

## Запуск

1. Склонируйте репозиторий:
   - `git clone <url-репозитория>`
2. Откройте файл `Лечебное меню.html` в браузере.

## Ограничения

- Проект не является медицинским назначением.
- Итоговое меню необходимо проверять с врачом.
- Для стабильной работы с PDF/DOCX нужен отдельный парсер или backend.
