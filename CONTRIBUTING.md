# Contributing

## Как вносить изменения

1. Создайте ветку от `main`.
2. Делайте небольшие осмысленные коммиты.
3. Проверяйте, что сайт открывается без ошибок в браузере.
4. В PR опишите:
   - что изменили;
   - зачем изменили;
   - как проверяли.

## Стиль

- По умолчанию используйте UTF-8.
- Сохраняйте читаемость HTML/CSS/JS.
- Не удаляйте предупреждение о том, что проект не заменяет врача.

## Проверка перед PR

- Ручной запуск `Лечебное меню.html`.
- Проверка кнопок:
  - генерация меню;
  - замена блюда;
  - сохранение/восстановление черновика;
  - скачивание JSON.
